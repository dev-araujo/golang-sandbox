# Golang Sandbox

Sandbox para estudos e experimentos com Go.

## Projetos

- **go-job:** [/go-job](./go-job)
- **Tecnologias:** Go, Gin, GORM, SQLite
- **Descrição:** API REST para gerenciamento de vagas de emprego com operações CRUD

---

## Estrutura

```
golang-sandbox/
├── README.md
└── go-job/
```

## Como usar

### Clonar o repositório

```bash
git clone https://github.com/dev-araujo/golang-sandbox.git
cd golang-sandbox
```

### Executar um projeto

```bash
cd go-job
go mod download
go run main.go
```

A API estará disponível em `http://localhost:8080`.

---

## Adicionar novos projetos

1. Crie uma nova pasta na raiz
2. Inicialize: `go mod init`
3. Adicione código e README
4. Atualize este README

**Nota:** Este repositório consolida projetos de estudo em Go.
